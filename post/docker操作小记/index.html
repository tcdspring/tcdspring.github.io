Docker操作小记

2019-06-04
docker

文章目录

在使用docker创建了容器后，那如何进入容器呢，下面记录一下自己常用的两种方法。

方法一 使用docker attach进入

1
2
3
4

sudo docker run -itd ubuntu:16.04 /bin/bash

// 输出一段字符串
// 比如说 245b07d79040514233c306acc48d7f372f2adad3a045448156aedf4c604b0c50

拿到输出的字符串，然后执行下面命令：

1

sudo docker attach 245b07d79

成功后会进入容器中，可以进行交互了。

方法二 使用docker exec进入

在docker1.3.x版本后提供了一个新的命令exec，用这种方式更简单一些

1
2

sudo docker ps
sudo docker exec -it 245b07d79 /bin/bash

还有可以通过ssh进入容器，只不过还得需要先进入容器中安装或者配置ssh，比较麻烦，所以就不是很推荐这种方式。
